Open Source Content Management Framework

LUTECE is a Java™ based CMS, Portal and Framework. It uses best of breed Open Source software libraries Spring, Ehcache, Freemarker, Lucene, ...

It provides a very modular architecture with a powerful assembling system based on Apache Maven.
